Everything You Need

Four validation layers plus seamless integrations for your entire development workflow.

AST Syntax Validation

Tree-sitter powered parsing catches malformed code in Python, JavaScript, and TypeScript before it runs.

Import Verification

Validates every import against PyPI and npm registries. Catches hallucinated packages instantly.

Signature Checking

Jedi-powered introspection validates function parameters, required args, and keyword arguments against real APIs.

LLM Output Parser

Extracts code blocks from ChatGPT/Claude markdown responses and validates each block independently.

Pre-commit Hook

Automatic validation on every git commit. Catches issues before they enter your codebase. SQLite cached.

VS Code Extension

Real-time inline diagnostics. Squiggly underlines for errors. Configurable on-save or on-change trigger.

4-Layer Validation Pipeline

Each layer catches different categories of AI hallucinations, from syntax errors to wrong function signatures.

Layer 1

Syntax

tree-sitter AST parsing catches malformed code

Layer 2

Imports

Verifies packages exist on PyPI & npm registries

Layer 3

Signatures

Validates function calls against real API signatures

Layer 4

Deprecation

Flags deprecated patterns with replacement suggestions

Fits Your Workflow

CLI, API server, pre-commit hooks, VS Code extension — use it however you work.

CLI Primary 
$ firewall check app.py
$ firewall check src/*.py --format json
$ firewall parse response.md
$ cat code.py | firewall check --stdin -l python
$ firewall parse --url https://gist.github.com/...
REST API FastAPI 
$ firewall serve --port 8000

$ curl -X POST localhost:8000/validate \
  -H "Content-Type: application/json" \
  -d '{"code": "import fakelib", "language": "python"}'
Pre-commit Hook Automatic 
# .pre-commit-config.yaml
repos:
  - repo: https://github.com/tranhoangtu-it/...
    rev: v0.1.0
    hooks:
      - id: firewall-check      # Python
      - id: firewall-check-js   # JS/TS

Get Started in 30 Seconds

terminal 
# Install
$ pip install -e "git+https://github.com/tranhoangtu-it/AI-Hallucination-Firewall.git#egg=hallucination-firewall"

# Validate a file
$ firewall check your_ai_code.py

# Validate LLM response
$ firewall parse chatgpt_response.md

# Start API server
$ firewall serve

# JSON output for CI/CD
$ firewall check --format json src/*.py
