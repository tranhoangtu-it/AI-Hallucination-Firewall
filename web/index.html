Frequently Asked Questions

Everything you need to know about using AI Hallucination Firewall.

Does it slow down my CI pipeline?

No, average validation is under 2 seconds per file. Parallel async checks ensure minimal overhead.

What about false positives?

Fail-open policy. Unknown functions are skipped, not flagged. We only report what we can verify is wrong.

Which languages are supported?

Python, JavaScript, TypeScript. More languages coming soon based on community feedback.

How is this different from ESLint/Pylint?

Those check code style. We check if AI-generated APIs actually exist in the real world.

Can I use it offline?

Syntax and signature checks work offline. Import checks need network for registry lookups (with caching).

Is it safe to use?

Read-only validation. Never executes your code. Restricted to allowlisted stdlib modules for safety.
